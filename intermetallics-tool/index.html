Intermetallics Tool

Experimental workflow tool for magnetization data and related calculations.

Overview

Intermetallics Tool is a Windows application designed for processing and analyzing magnetization datasets such as MT, MH, and hysteresis measurements.

Beyond data processing, it integrates essential utilities like a periodic table, unit conversion, and composition calculations to streamline experimental workflows.

Core Processing

  • Load MT, MH, and hysteresis data files
  • Generate M(T) and dM/dT outputs
  • Arrott plot generation
  • Magnetocaloric effect (MCE) calculations
  • Organized and export-ready output files

Utilities

  • Interactive periodic table with element properties
  • Electronic configuration display for all elements
  • Scientific unit converter (magnetism, energy, temperature, mass, volume, pressure)
  • Formula-based weight calculator for multi-element systems

Design Philosophy

  • Focused on reducing data processing time, not replacing analysis
  • Minimal and distraction-free interface
  • Offline operation with no external dependencies
  • Built for practical lab use and repeatability

System Requirements

  • Windows 10 or later
  • No additional dependencies required
