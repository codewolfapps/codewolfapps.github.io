Intermetallics Tool

Data processing tool for magnetization measurements.

Overview

Intermetallics Tool is a Windows application designed for analyzing magnetization datasets such as MT, MH, and hysteresis measurements.

It provides a simple graphical interface to process experimental data and generate useful outputs for research and analysis.

Features

  • Load MT, MH, and hysteresis data files
  • Generate M(T) and dM/dT plots
  • Arrott plot generation
  • Magnetocaloric effect (MCE) calculations
  • Organized output files

System Requirements

  • Windows 10 or later
  • No additional dependencies required
